============================
tryp Python Reporting Tool
============================

-----------------
Setting up Tryp
-----------------

* Manually install numpy:

.. code-block :: python

    pip install numpy

* Do installation. This will pull down all the requirements 

.. code-block :: python

    python setup.py install
    
* Running tests

.. code-block :: python

   python setup.py nosetests
